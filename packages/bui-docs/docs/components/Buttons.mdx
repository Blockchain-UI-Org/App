# Button

Buttons allow users to take actions, and make choices, with a single tap.

## Basic Buttons

The Button comes with four variants: contained (default), text, outlined and soft.

<DisplayExample row justifyContent="center">

<Button label="Contained" />
<HorizontalSpacer />
<Button label="Text" variant="text" />
<HorizontalSpacer />
<Button label="Outlined" variant="outlined" />

</DisplayExample>

```jsx
<Button label="Contained" />
<Button label="Text" variant="text" />
<Button label="Outlined" variant="outlined" />
```

## Text Buttons

Text buttons are typically used for less-pronounced actions, including those located: in dialogs, in cards. In cards, text buttons help maintain an emphasis on card content

<DisplayExample row justifyContent="center">
  <Button variant="text" label="Primary" />
  <HorizontalSpacer />
  <Button variant="text" disabled label="Disabled" />
  <HorizontalSpacer />
  <Button variant="text" href="#text-buttons" label="Link" />
</DisplayExample>

```jsx
<Button variant="text" label="Primary" />
<Button variant="text" disabled label="Disabled" />
<Button variant="text" href="#text-buttons" label="Link" />
```

## Contained Buttons

Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app.

<DisplayExample row justifyContent="center">
  <Button label="Contained" />
  <HorizontalSpacer />
  <Button disabled label="Disabled" />
  <HorizontalSpacer />
  <Button href="#contained-buttons" label="Link" />
</DisplayExample>

```jsx
<Button  label="Contained" />
<Button  disabled label="Disabled" />
<Button  href="#contained-buttons" label="Link" />
```

## Outlined Buttons

Outlined buttons are medium-emphasis buttons. They contain actions that are important but aren't the primary action in an app.

<DisplayExample row justifyContent="center">
  <Button variant="outlined" label="Outlined" />
  <HorizontalSpacer />
  <Button variant="outlined" disabled label="Disabled" />
  <HorizontalSpacer />
  <Button variant="outlined" href="#outlined-buttons" label="Link" />
</DisplayExample>

```jsx
<Button variant="outlined" label="Outlined" />
<Button variant="outlined" disabled label="Disabled" />
<Button variant="outlined" href="#outlined-buttons" label="Link" />
```

## Soft Buttons

Soft buttons are light-emphasis buttons. They contain actions that are important but aren't the primary action in an app.
