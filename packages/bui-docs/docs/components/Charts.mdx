# Charts

Charts Allow to display and view data for analysis

## Basic Area Chart

<DisplayExample>
  <AreaChart
    labels={MonthlyData.map((item) => item.month)}
    formatXAxis={(val, t, opts) => {
      return val.slice(0, 3);
    }}
    formatYAxis={(val) => {
      return val.toFixed(0);
    }}
    series={[
      { name: "Total Income", color: "primary500", data: MonthlyData.map((item) => item.totalIncome) },
    ]}
  />
</DisplayExample>

```jsx
<AreaChart
    labels={["January", "February", ... /* Rest of months  */]}
    formatXAxis={(val, t, opts) => {
      return val.slice(0, 3);

      return val.toFixed(0);
    }}

    ]}
  />
```
