# Table

It is a responsive table component built on the top of styled-components. It comes with the feature of searching, customizing columns,
defining custom renders and paginations.

## Examples

There are multiple configurations to apply on table. It takes columns and data as a primary arguments.

### Simple Table

This is a minimal table example with sample data.

```jsx
<Table columns={COLUMNS} data={DATA} options={OPTIONS} />
<Table columns={COLUMNS} data={DATA} options={{
  search: true,
  pagination: {
    enable: true,
    rowsPerPage: {
        enable: true,
        values: [5, 10, 15, 20, 30, 50],
        defaultValue: 5,
    },
  },
}} />
```

### Table without pagination

```jsx
<Table
  columns={COLUMNS}
  data={DATA}
  options={{
    search: true,
    pagination: {
      enable: false,
    },
  }}
/>
```

### Table without search

```jsx
<Table
  columns={COLUMNS}
  data={DATA}
  options={{
    search: false,
    pagination: {
      enable: true,
    },
  }}
/>
```
