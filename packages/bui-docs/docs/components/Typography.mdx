# Typography

The Typography component makes it easy to apply a default set of font weights and sizes in your application.

Use typography to present your design and content as clearly and efficiently as possible.


## Heading Component

Heading Component is for various headings in your application.

<Heading as="h2">This is a H2 Heading</Heading>
<Heading as="h3">This is a H3 Heading</Heading>
<Heading as="h4">This is a H4 Heading</Heading>
<Heading as="h5">This is a H5 Heading</Heading>
<Heading as="h6">This is a H6 Heading</Heading>


</DisplayExample>


## Subtitle Component

<Subtitle variant="subtitle2" >This is a Subtitle2 Variant</Subtitle>


</DisplayExample>

```jsx
<Subtitle variant="subtitle1" >This is a Subtitle1 Variant</Subtitle>
```

## Typography Component

This is the base component from which Paragraph, Heading, Subtitle Components are created.

All Typography Component have the following Props:

- fontWeight
- fontSize
- lineHeight
- color
- marginTop
- marginBottom
- fontFamily

<DisplayExample alignItems="flex-start">
<Typography>
    This is Default Typography Component
</Typography>
<Typography fontSize={30} fontWeight={400} color="#99e" marginTop="1rem" marginBottom={"1rem"} >
    This is Customised Typography Component
</Typography>
</DisplayExample>

```jsx
<Typography>
    This is Default Typography Component
</Typography>
<Typography fontSize={30} marginTop="1rem" fontWeight={400} color="#99e" marginBottom="1rem" >
    This is Customised Typography Component
</Typography>
```
