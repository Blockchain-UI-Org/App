# Notifications

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

## Examples

Notification are available for small messages of text. For proper styling, use one of the five available Notification

### Simple Notification

This is a minimal Alert with 4 Statuses i.e. error, warning, success and info.

<DisplayExample>

<Notification rootStyle={{position: "relative", top: "initial", left: "initial", transform: "initial"}}  message="A Simple Info Notification"  />
<Spacer/>
<Notification rootStyle={{position: "relative", top: "initial", left: "initial", transform: "initial"}} type="error"  message="A Simple Error Notification"  />
<Spacer/>
<Notification rootStyle={{position: "relative", top: "initial", left: "initial", transform: "initial"}} type="warning"  message="A Simple Warning Notification"  />
<Spacer/>
<Notification rootStyle={{position: "relative", top: "initial", left: "initial", transform: "initial"}} type="success"  message="A Simple Success Notification"  />


</DisplayExample>

```jsx
<Notification  message="A Simple Info Alert"  />

<Notification  type="error"  message="A Simple Error Notification"  />

<Notification  type="warning"  message="A Simple Warning Notification"  />

<Notification  type="success"  message="A Simple Success Notification"  />
```
